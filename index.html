Cola's Enumeration

Cola's Enumeration for Naming Large Numbers: An Extension of the Conway-Wechsler System

Do you like counting?

This is a counting machine. It names lots of numbers that did not have names. It also shortens the names that are too long. It can count forever! Why? Well, because it was fun to make.

Features

-Enumerates numbers as high as 10 to the power of ten thousand.

-Includes a console application for Conway's illion Converter

-Includes tables and a readme for reference
